3 条命令，从 YouTube 到成片

Mac / Apple Silicon 友好 · 一键安装依赖 · MIT 协议

# 1. 装 skill
$ git clone https://github.com/KevPH2026/kev-youtube-video-clone-translate
$ bash scripts/install-deps.sh
# 2. 给 Claude 一个 YouTube 链接
$ claude "把这个翻译成中文，克隆原说话人音色"
# 3. 30 分钟后你的 ~/Downloads 多一个 dub_v5.mp4
